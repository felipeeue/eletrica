Transformação Y-Delta e Delta-Y

A transformação Y-Delta e Delta-Y são utilizadas para simplificar circuitos com resistores dispostos em estrela (Y) ou triângulo (Delta). Essas transformações são essenciais para facilitar cálculos de resistências equivalentes em circuitos mais complexos.

Transformação Y para Delta: A transformação de resistores de uma configuração estrela (Y) para uma configuração triângulo (Delta) pode ser feita utilizando as fórmulas abaixo. Essa transformação é útil quando você precisa encontrar as resistências equivalentes para análise do circuito.

Ra = (R1 * R2) / (R1 + R2 + R3)
Rb = (R2 * R3) / (R1 + R2 + R3)
Rc = (R1 * R3) / (R1 + R2 + R3)

Transformação Delta para Y: Para transformar um circuito Delta (triângulo) em Y (estrela), usamos as seguintes fórmulas:

R1 = (Ra * Rb) / (Ra + Rb + Rc)
R2 = (Rb * Rc) / (Ra + Rb + Rc)
R3 = (Rc * Ra) / (Ra + Rb + Rc)

Exemplos:

Exemplo 1: Se R1 = 10Ω, R2 = 20Ω e R3 = 30Ω, calcule as resistências correspondentes na configuração Delta utilizando as fórmulas mencionadas.

  • Ra = (10 * 20) / (10 + 20 + 30) = 200 / 60 = 3.33Ω
  • Rb = (20 * 30) / (10 + 20 + 30) = 600 / 60 = 10Ω
  • Rc = (10 * 30) / (10 + 20 + 30) = 300 / 60 = 5Ω

Exemplo 2: Se R1 = 15Ω, R2 = 25Ω e R3 = 35Ω, calcule as resistências na configuração Delta.

  • Ra = (15 * 25) / (15 + 25 + 35) = 375 / 75 = 5Ω
  • Rb = (25 * 35) / (15 + 25 + 35) = 875 / 75 = 11.67Ω
  • Rc = (15 * 35) / (15 + 25 + 35) = 525 / 75 = 7Ω

Exemplo 3: Se R1 = 5Ω, R2 = 10Ω e R3 = 15Ω, calcule as resistências na configuração Delta.

  • Ra = (5 * 10) / (5 + 10 + 15) = 50 / 30 = 1.67Ω
  • Rb = (10 * 15) / (5 + 10 + 15) = 150 / 30 = 5Ω
  • Rc = (5 * 15) / (5 + 10 + 15) = 75 / 30 = 2.5Ω

Exemplo 4: Se R1 = 8Ω, R2 = 12Ω e R3 = 18Ω, calcule as resistências na configuração Delta.

  • Ra = (8 * 12) / (8 + 12 + 18) = 96 / 38 = 2.53Ω
  • Rb = (12 * 18) / (8 + 12 + 18) = 216 / 38 = 5.68Ω
  • Rc = (8 * 18) / (8 + 12 + 18) = 144 / 38 = 3.79Ω

Exemplo 5: Se R1 = 9Ω, R2 = 16Ω e R3 = 24Ω, calcule as resistências na configuração Delta.

  • Ra = (9 * 16) / (9 + 16 + 24) = 144 / 49 = 2.94Ω
  • Rb = (16 * 24) / (9 + 16 + 24) = 384 / 49 = 7.84Ω
  • Rc = (9 * 24) / (9 + 16 + 24) = 216 / 49 = 4.41Ω

Ponte de Wheatstone

A Ponte de Wheatstone é um dispositivo utilizado para medir resistências desconhecidas. Quando a ponte está em equilíbrio, a razão entre os resistores conhecidos é igual à razão entre o resistor desconhecido e o resistor de referência.

Método de Cálculo:

Rx = (R2 * R3) / R1

Exemplos:

Exemplo 1: Se R1 = 10Ω, R2 = 20Ω e R3 = 30Ω, calcule Rx.

  • Rx = (20 * 30) / 10 = 600 / 10 = 60Ω

Exemplo 2: Se R1 = 15Ω, R2 = 25Ω e R3 = 35Ω, calcule Rx.

  • Rx = (25 * 35) / 15 = 875 / 15 = 58.33Ω

Exemplo 3: Se R1 = 5Ω, R2 = 10Ω e R3 = 15Ω, calcule Rx.

  • Rx = (10 * 15) / 5 = 150 / 5 = 30Ω

Exemplo 4: Se R1 = 12Ω, R2 = 18Ω e R3 = 24Ω, calcule Rx.

  • Rx = (18 * 24) / 12 = 432 / 12 = 36Ω

Exemplo 5: Se R1 = 8Ω, R2 = 16Ω e R3 = 32Ω, calcule Rx.

  • Rx = (16 * 32) / 8 = 512 / 8 = 64Ω

Teorema da Superposição

O Teorema da Superposição afirma que a resposta total de um circuito com múltiplas fontes é a soma das respostas causadas por cada fonte individualmente. Este teorema é aplicável a circuitos lineares.

Exemplos:

Exemplo 1: Se V1 = 5V e V2 = 3V, qual é a tensão total (Vt) no circuito?

  • Vt = 5V + 3V = 8V

Exemplo 2: Se V1 = 10V e V2 = 4V, qual é a tensão total (Vt)?

  • Vt = 10V + 4V = 14V

Exemplo 3: Se V1 = 6V e V2 = 2V, qual é a tensão total (Vt)?

  • Vt = 6V + 2V = 8V

Exemplo 4: Se V1 = 8V e V2 = 1V, qual é a tensão total (Vt)?

  • Vt = 8V + 1V = 9V

Exemplo 5: Se V1 = 7V e V2 = 5V, qual é a tensão total (Vt)?

  • Vt = 7V + 5V = 12V

Teoremas: Proporcionalidade, LKC e LKT

O Teorema da Proporcionalidade descreve a relação direta entre grandezas proporcionais. As Leis de Kirchhoff permitem determinar as correntes e tensões em circuitos.

Exemplos de LKC e LKT:

Exemplo 1 (LKC): Se três correntes entram em um nó (I1 = 3A, I2 = 2A e I3 = 1A), qual é a corrente que sai?

  • ΣI = 3A + 2A + 1A = 6A, portanto, a corrente que sai é 6A.

Exemplo 2 (LKC): Se as correntes entrando em um nó são I1 = 5A, I2 = 3A e I3 = 7A, qual é a corrente de saída?

  • ΣI = 5A + 3A + 7A = 15A, a corrente de saída é 15A.

Exemplo 3 (LKT): Em um circuito fechado, se as tensões são V1 = 5V, V2 = 3V, V3 = 4V, qual é a soma total das tensões?

  • ΣV = 5V + 3V + 4V = 12V

Exemplo 4 (LKT): Em um circuito fechado, se as tensões são V1 = 7V, V2 = 2V, V3 = 8V, qual é a soma total das tensões?

  • ΣV = 7V + 2V + 8V = 17V

Exemplo 5 (LKC e LKT): Se você tem dois nós com correntes I1 = 4A, I2 = 6A e tensões V1 = 5V e V2 = 10V, calcule as variáveis aplicando LKC e LKT.

  • ΣI = 4A + 6A = 10A
  • ΣV = 5V + 10V = 15V

Atividade 2: Cálculos de Resistência e Corrente

2.1) Calcular a resistência de um fio de alumínio de 200m de comprimento e 2mm² de seção.

Passo 1: Fórmula da resistência de um fio condutor

A fórmula para calcular a resistência de um fio condutor é:

R = ρ × (L / A)

Onde:

  • R = resistência (Ω)
  • ρ = resistividade do material (Ω·m)
  • L = comprimento do fio (m)
  • A = área da seção transversal (m²)

Dados:

  • ρ (resistividade do alumínio) = 2,8 × 10-8 Ω·m
  • L = 200m
  • A = 2mm² = 2 × 10-6 m²

Substituindo os valores na fórmula:

R = 2,8 × 10-8 × (200 / 2 × 10-6)

R = 2,8 × 10-8 × 108 = 2,8 Ω

Resposta: A resistência do fio de alumínio é 2,8 Ω.

2.2) Um fio de cobre tem 2mm de diâmetro, aplicando-se uma tensão de 20V ao fio, resulta uma corrente de 5A. Qual é o comprimento do fio?

Passo 1: Fórmula para calcular a resistência de um fio condutor

A resistência de um fio é dada pela fórmula:

R = ρ × (L / A)

Onde:

  • ρ (resistividade do cobre) = 1,7 × 10-8 Ω·m
  • L = comprimento do fio (a ser calculado)
  • A = área da seção transversal do fio

Passo 2: Área da seção transversal (A)

O fio tem diâmetro de 2mm, então o raio (r) é 1mm = 1 × 10-3 m.

A área da seção transversal de um fio circular é dada por:

A = π × r²

A = π × (1 × 10-3)² ≈ 3,1416 × 10-6 m²

Passo 3: Calcular a resistência (R) do fio

Usando a Lei de Ohm, sabemos que:

R = V / I

Onde:

  • V = 20V
  • I = 5A

Logo, a resistência é:

R = 20 / 5 = 4 Ω

Passo 4: Calcular o comprimento do fio (L)

Agora, substituímos os valores conhecidos na fórmula da resistência do fio:

R = ρ × (L / A)

4 = (1,7 × 10-8) × (L / 3,1416 × 10-6)

4 = (1,7 × 10-8 × L) / 3,1416 × 10-6

L = (4 × 3,1416 × 10-6) / 1,7 × 10-8 ≈ 7,4 × 103 m

Resposta: O comprimento do fio é aproximadamente 7400 metros.

2.3) A resistência de um condutor é 25kΩ. Calcular:

a) Sua condutância

A condutância (G) é o inverso da resistência (R), ou seja:

G = 1 / R

Substituindo o valor de R (25kΩ = 25.000Ω):

G = 1 / 25.000 = 0,00004 S (Siemens)

b) Corrente em mA quando submetido a 10V

A corrente (I) pode ser calculada pela Lei de Ohm:

I = V / R

Substituindo os valores de V (10V) e R (25.000Ω):

I = 10V / 25.000Ω = 0,0004 A = 0,4mA

Resposta: A corrente é 0,4mA.

Atividade 3: Chuveiro Elétrico

3) Um chuveiro tem as especificações 3500W/220V. Calcular:

a) Intensidade da corrente

A potência elétrica (P) é dada pela fórmula:

P = V × I

Onde:

  • P = 3500W
  • V = 220V
  • I = intensidade da corrente (a ser calculada)

Substituindo os valores:

I = P / V = 3500 / 220 ≈ 15,91 A

Resposta: A intensidade da corrente é aproximadamente 15,91A.

b) Valor da resistência

A resistência (R) pode ser calculada pela Lei de Ohm:

R = V / I

Substituindo os valores:

R = 220V / 15,91A ≈ 13,84 Ω

Resposta: A resistência é aproximadamente 13,84Ω.

c) Energia gasta durante 30 minutos, em kWh

A energia (E) é dada pela fórmula:

E = P × t

Onde:

  • P = 3500W
  • t = tempo em horas = 30 minutos = 0,5 horas

Substituindo os valores:

E = 3500W × 0,5h = 1750 Wh = 1,75 kWh

Resposta: A energia gasta é 1,75 kWh.

4) Quatro resistores R1 = 101Ω, R2 = 200Ω, R3 = 40Ω e R4 = 80Ω são ligados em série. Sabendo-se que a tensão em R3 é de 20V. Pede-se:

a) Resistor equivalente (Req)

Em uma associação em série, a resistência equivalente (Req) é a soma das resistências dos resistores:

Req = R1 + R2 + R3 + R4

Substituindo os valores dados:

Req = 101Ω + 200Ω + 40Ω + 80Ω = 421Ω

Resposta: A resistência equivalente é 421Ω.

b) Tensão aplicada na associação

Sabemos que a tensão em R3 é de 20V. Para calcular a tensão total da associação, utilizamos a Lei de Ohm:

V = I × Req

Primeiro, precisamos calcular a corrente (I) que passa pelos resistores. Usamos a Lei de Ohm para R3:

I = V_R3 / R3

Substituindo os valores:

I = 20V / 40Ω = 0,5A

Agora, podemos calcular a tensão total aplicada à associação:

V_total = I × Req = 0,5A × 421Ω = 210,5V

Resposta: A tensão aplicada na associação é 210,5V.

c) Potência dissipada na associação

A potência dissipada (P) em uma associação de resistores pode ser calculada usando a fórmula:

P = V_total × I

Substituindo os valores:

P = 210,5V × 0,5A = 105,25W

Resposta: A potência dissipada na associação é 105,25W.

5) Quatro resistores R1 = 51Ω, R2 = 40Ω, R3 = 60Ω e R4 = 120Ω são ligados em paralelo. Sabendo-se que I4 = 0,5A, determinar:

a) Resistência equivalente (Req)

Em uma associação de resistores em paralelo, a resistência equivalente (Req) é dada pela fórmula:

1 / Req = 1 / R1 + 1 / R2 + 1 / R3 + 1 / R4

Substituindo os valores dados:

1 / Req = 1 / 51 + 1 / 40 + 1 / 60 + 1 / 120

1 / Req ≈ 0,01961 + 0,025 + 0,01667 + 0,00833 ≈ 0,06961

Agora, invertendo o valor para encontrar Req:

Req ≈ 1 / 0,06961 ≈ 14,38Ω

Resposta: A resistência equivalente é aproximadamente 14,38Ω.

b) Tensão aplicada na associação e corrente em todos os resistores

Sabemos que I4 = 0,5A, e os resistores estão em paralelo. Como todos os resistores estão submetidos à mesma tensão, podemos calcular a tensão aplicada (V) usando a Lei de Ohm para o resistor R4:

V = I4 × R4

Substituindo os valores:

V = 0,5A × 120Ω = 60V

A tensão aplicada na associação é 60V.

Agora, para calcular a corrente em cada resistor, utilizamos a Lei de Ohm (I = V / R) para cada um deles:

Para R1:

I1 = 60V / 51Ω ≈ 1,18A

Para R2:

I2 = 60V / 40Ω = 1,5A

Para R3:

I3 = 60V / 60Ω = 1A

Para R4 (já dado):

I4 = 0,5A

Resposta: A corrente em cada resistor é:

  • I1 ≈ 1,18A
  • I2 = 1,5A
  • I3 = 1A
  • I4 = 0,5A

c) Potência dissipada nos resistores e no equivalente

A potência dissipada em cada resistor pode ser calculada usando a fórmula:

P = V² / R

Para cada resistor:

Para R1:

P1 = 60² / 51 ≈ 70,59W

Para R2:

P2 = 60² / 40 = 90W

Para R3:

P3 = 60² / 60 = 60W

Para R4:

P4 = 60² / 120 = 30W

Agora, a potência total dissipada é a soma das potências dissipadas por cada resistor:

P_total = P1 + P2 + P3 + P4 ≈ 70,59W + 90W + 60W + 30W = 250,59W

Para calcular a potência dissipada pelo resistor equivalente, usamos a fórmula:

P_eq = V² / Req

Substituindo os valores:

P_eq = 60² / 14,38 ≈ 250,59W

Resposta: A potência dissipada nos resistores é 250,59W, e a potência dissipada no resistor equivalente é a mesma: 250,59W.
